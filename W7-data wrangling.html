Introduction

In this workshop, we learn how to wrangle data using the Tidyverse packages in R.
The most commonly used packages for data wrangling include dplyr and tidyr.
These packages help transform, organize, and clean data so it is easier to analyze.

Tidying Data with tidyr

pivot_longer()

The pivot_longer() function converts data from wide format to long format.

library(tidyr)

long_data <- mtcars %>%
  pivot_longer(
    cols = mpg:hp,
    names_to = "variable",
    values_to = "value"
  )

separate()

The separate() function splits a column into multiple columns.

df <- data.frame(name = c("John-Doe", "Jane-Smith"))

separate(df, name, into = c("first", "last"), sep = "-")
  first  last
1  John   Doe
2  Jane Smith

unite()

The unite() function combines multiple columns into one.

df2 <- data.frame(
  first = c("John","Jane"),
  last = c("Doe","Smith")
)

unite(df2, "full_name", first, last, sep = " ")
   full_name
1   John Doe
2 Jane Smith

Example of Grouped Analysis

We can combine multiple functions together.

mtcars %>%
  group_by(cyl) %>%
  summarize(
    avg_mpg = mean(mpg),
    avg_hp = mean(hp)
  )
# A tibble: 3 × 3
    cyl avg_mpg avg_hp
  <dbl>   <dbl>  <dbl>
1     4    26.7   82.6
2     6    19.7  122. 
3     8    15.1  209.

Conclusion

The Tidyverse provides powerful tools for data wrangling and transformation.
Functions from tidyr help tidy and restructure datasets, while functions from dplyr allow us to filter, select, and transform data efficiently.
These techniques are essential for preparing data before performing analysis or visualization.
